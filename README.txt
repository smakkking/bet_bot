как работает ПРОЦЕСС
каждый момент времени группа чекается на выход нового поста
если в новом посте есть картинка, то она прогоняется через распознавалку текста, а потом через шаблон ставки
если это картинка ставки, то через букмекерку проходит ставка
про выбор бк:
нужно импортировать все файлы bk в new_script, а потом передавать в качестве аргумента функцию ставки!

для каждой группы VK в модуле должно быть написано:
1) парсинг текста ставки(parse_bet)
2) основной скрипт, выполняющий ПРОЦЕСС(main_script)

для каждой бк в модуле должно быть написано:
1) поиск ставки(search_bet)
2) процесс ставки(bet)
